Pour utiliser NPM, installer node

- Créer un nouveau dossier et l'ouvrir dans VSCode
- Ouvrir un terminal et démarrer un projet Git :
git init

- Démmarrer un projet npm : 
npm init -y

- Installer webpack (qui est un node module)
npm install --save-dev webpack
